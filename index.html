Natural Language v. Boolean Example

This is a vibe coded application using Google AI studio to demonstrate differences between legal database searches. This application is for academic purposes only and is not intended to give legal or professional advice.

1. Natural Language

Simulates semantic relevance. It parses individual keywords, ignores common stop words, and ranks results by keyword density and location.

2. Terms & Connectors

Simulates classic Boolean syntax. Supports strict operators (AND, OR, NOT) and double quotes ("") for exact phrases.

3. Filtered Search

Demonstrates metadata-first search. Restricts keyword search scope by categorizing document attributes before querying content.
